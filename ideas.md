## Idea A — "Clean Lines, Bright Skies" (Chosen)

**Design Movement:** Friendly British Service Brand — think Dyson meets a local tradesperson. Clean, trustworthy, approachable.

**Core Principles:**
1. Sky blue as the dominant canvas — warm and airy, not corporate
2. Navy for authority and trust (headings, nav)
3. Orange as the action colour — every CTA pops in amber
4. Green reserved for the phone number and "free delivery" badge — mirrors the flyer exactly

**Color Philosophy:** The flyer already tells us the story. Sky blue = freshness and cleanliness. Navy = reliability. Orange = energy and urgency. Green = value and savings. We honour this palette faithfully so the website feels like a natural extension of the printed material Sandra already hands out.

**Layout Paradigm:** Mobile-first card stack. On mobile: full-width sections stacked vertically with generous padding. On desktop: asymmetric two-column layouts where text sits left and a visual sits right. The hero is a full-bleed sky-blue panel with a wave/cloud SVG divider at the bottom — echoing the flyer's cloud motif.

**Signature Elements:**
1. Cloud-wave SVG dividers between sections — directly referencing the flyer's cloud shapes
2. The phone number displayed in large green bold type, always visible in the sticky header on mobile
3. Orange pill/badge for promotional callouts ("Free delivery on orders over £15")

**Interaction Philosophy:** Tap-friendly. Every CTA is a minimum 48px touch target. Phone number is a `tel:` link so mobile users can dial with one tap. Smooth scroll between sections.

**Animation:** Subtle fade-up entrance animations on section entry (Framer Motion). The hero truck illustration gently slides in from the left on load. No excessive motion — this is a local service business, not a tech startup.

**Typography System:**
- Headings: Nunito (rounded, friendly, matches the flyer's rounded letterforms) — Bold 700/800
- Body: Open Sans — Regular 400, Medium 500
- Phone number: Nunito ExtraBold 800, oversized, green

## Idea B — "Laundry Day Warmth"

**Design Movement:** Warm Domestic Minimalism — cream backgrounds, soft shadows, hand-drawn accents.

**Core Principles:**
1. Cream/off-white base with warm undertones
2. Navy as the only dark colour
3. Thin line illustrations of garments as decorative elements
4. Generous whitespace

**Color Philosophy:** Step away from the bright blue and lean into warmth — cream, linen, soft amber. The blue appears only as an accent.

**Layout Paradigm:** Centered editorial layout with large type and minimal imagery. Feels like a premium laundry brand.

**Signature Elements:** Thin SVG garment icons, serif display headings, handwritten-style accent text.

**Interaction Philosophy:** Slow, considered scrolling. Parallax on the hero image.

**Animation:** Slow fade and gentle parallax. Garment icons float slightly on hover.

**Typography System:** Playfair Display for headings, Lato for body.

## Idea C — "Bold Local Business"

**Design Movement:** High-contrast utility brand — bold blocks of colour, no gradients, strong grid.

**Core Principles:**
1. Navy blocks alternating with white blocks
2. Orange used aggressively for headings and borders
3. No rounded corners — sharp, confident
4. Large numbers and statistics

**Color Philosophy:** Confidence and directness. This is a working business, not a lifestyle brand.

**Layout Paradigm:** Full-width alternating colour bands. Each section is a distinct colour block.

**Signature Elements:** Bold section numbers, thick orange borders, oversized phone number.

**Interaction Philosophy:** Fast, direct. No animations — just clean navigation.

**Animation:** None intentional — speed is the message.

**Typography System:** Oswald for headings, Roboto for body.
